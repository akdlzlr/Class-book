<조건문>

if의 조건식의 필수 조건  =   조건식이 부울변수여야 한다
a=1 는 조건이 되지 못한다.
a==1 만 조건이 될 수 있다.

Math.random()의 식에 원하는 갯수(구간)를 곱한다.
Math.random()*10  =  10개
Math.random()*6   =   6개

Math.random()의 식에 원하는 숫자를 더한다. 시작점을 설정
Math.random()*10+1   =  1부터 시작
Math.random()*10+2   =  2부터 시작


switch의 조건식의 필수 조건   =  실수는 오차 때문에 사용하지 않는다. 정수나 문자열이 와야한다.


if의 else와 switch의 default는 생략 가능하다.
